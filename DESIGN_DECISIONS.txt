================================================================================
DESIGN DECISIONS LOG — METHOD EVALUATION
================================================================================
Tracks decisions made during benchmarking to evaluate the superiority or
inferiority of different approaches. Each decision documents the rationale
and the comparison it enables.
Last updated: 2026-03-01

================================================================================
D1: LOSS FUNCTION COMPARISON
================================================================================
Decision: Compare Dice loss vs focal loss for fine-tuning
Rationale: Both are common for segmentation; focal loss down-weights easy pixels
Result: Focal loss outperformed Dice loss 5.4x (mito) and 2.4x (membrane)
Implication: Focal loss is the default for all subsequent experiments

================================================================================
D2: DOMAIN ADAPTATION vs SUPERVISED FINE-TUNING
================================================================================
Decision: Compare unadapted, DA-only, fine-tuned, and DA+FT
Rationale: DA could help bridge cryo-ET → retinal TEM domain gap without labels
Result: DA alone = Dice 0.000; DA+FT marginal improvement over FT alone
Implication: Supervised fine-tuning is essential; DA adds negligible value

================================================================================
D3: PRETRAINED 3D MODEL ZERO-SHOT EVALUATION
================================================================================
Decision: Run pretrained SynapseNet 3D models (mitochondria, vesicles_3d,
         compartments, ribbon, active_zone) zero-shot on full 7000x7000 montages
Rationale: Establishes baseline — do pretrained cryo-ET models transfer at all?
Result: All models produce essentially empty predictions (fg < 0.02%)
Implication: Confirms complete domain gap; fine-tuning is mandatory

================================================================================
D4: micro-SAM ZERO-SHOT
================================================================================
Decision: Test micro-SAM (vit_b_em_organelles, AMG) as a training-free baseline
Rationale: Foundation model may generalize better than task-specific pretrained
Result: Non-zero but low Dice (mito ~0.05, membrane ~0.03)
Implication: Only method to produce non-zero results without training, but
            far below practical utility

================================================================================
D5: TTA TOGGLE (TEST-TIME AUGMENTATION)
================================================================================
Decision: Evaluate each model with and without 7 geometric TTA augmentations
Rationale: TTA averages predictions over augmented views, may smooth errors
Result: Improved mito (part of best ensemble); no improvement for membrane
Implication: TTA benefit is structure-dependent; include as optimization toggle

================================================================================
D6: ENSEMBLE vs SINGLE MODEL
================================================================================
Decision: Compare single best model vs max-voting ensemble of 3 models (mito)
Rationale: Ensembling combines complementary errors across model variants
Result: Ensemble + TTA reached Dice 0.869 (mito), best overall
         Single model sufficient for membrane (0.892)
Implication: Ensemble useful when multiple model variants available

================================================================================
D7: THRESHOLD OPTIMIZATION
================================================================================
Decision: Sweep binarization threshold 0.05-0.90 for every model configuration
Rationale: Default threshold (0.50) is rarely optimal for imbalanced structures
Result: Optimal thresholds vary widely (mito 0.35, membrane 0.40, ribbon 0.80,
        vesicles 0.05)
Implication: Threshold sweep is mandatory post-training step

================================================================================
D8: FOREGROUND WEIGHTING PER STRUCTURE
================================================================================
Decision: Assign per-structure foreground weights in focal loss
         (mito=5, membrane=2, ribbon=50, vesicles=20)
Rationale: Structures have very different foreground ratios (membrane ~15%,
          ribbon <0.1%)
Result: Appropriate weighting critical for sparse structures (ribbon)
Implication: Weight must be tuned per structure based on class imbalance

================================================================================
D9: RIBBON-PROXIMITY GUIDED VESICLE SEGMENTATION
================================================================================
Decision: Train vesicle model on ribbon-proximal labels only; at inference,
         mask predictions to within 50px of predicted ribbon
Rationale: 100% of GT vesicles lie within 150px of ribbon in our annotations;
          unfiltered predictions have high false-positive rate
Result: Nearly doubled vesicle Dice (0.105 → 0.206)
Implication: Anatomical priors as spatial constraints improve performance
             when biological relationship is known

================================================================================

Decision: Test pretrained 3D ribbon model both with and without vesicle
         segmentation passed as context (extra_segmentation parameter)
Rationale: SynapseNet ribbon pipeline optionally uses vesicle predictions for
          post-processing. Compare: (a) ribbon alone, (b) ribbon + vesicle context
Result: [PENDING — to be evaluated]

================================================================================
D11: POST-HOC 3D CONSISTENCY ON 2D PREDICTIONS
================================================================================
Decision: Apply 3D consistency enforcement (3D connected components, instance
         tracking, z-smoothing) as independent toggle on all 2D predictions
Rationale: Measures how much volumetric consistency alone improves 2D predictions,
          providing a fair comparison against native 3D inference
Methods:
  a) 3D connected components: stack masks, filter by volume
  b) Instance tracking: link instances across z by overlap
  c) Z-smoothing: Gaussian filter on probability maps along z, re-threshold
Result: [PENDING — to be evaluated with extended dataset]
Implication: If 2D + post-hoc matches 3D, avoids expensive 3D training

================================================================================
D12: 2.5D vs 2D vs 3D ARCHITECTURE COMPARISON
================================================================================
Decision: Benchmark same method matrix across 3 architecture classes:
         2D U-Net, 2.5D (3-channel adjacent slices), 3D AnisotropicUNet
Rationale: Isolates the effect of z-context on segmentation quality
Requirements: >= 8 annotated serial sections (current 3 insufficient for 3D)
Result: [PENDING — next dataset]
Implication: Determines optimal architecture for each structure type

================================================================================
D13: MORPHOLOGICAL POST-PROCESSING PARAMETERS
================================================================================
Decision: Apply opening(disk(2)) + min_area filtering per structure
Rationale: Opening removes thin noise; min_area removes tiny false positives
Parameters: mito 5000px, membrane 50000px, ribbon 100px, vesicles 20px
Result: Improves visual quality; parameters tuned empirically
Implication: Post-processing parameters are structure-specific

================================================================================
D14: FULL MONTAGE GENERALIZATION
================================================================================
Decision: Run best models on full 7000x7000 montages (m0-m3), not just
         annotated crop regions
Rationale: Tests whether models generalize beyond training crop (~3.6% of field)
Result: Models produce plausible predictions across full montages;
        m0/m1 richer in mito, m2/m3 richer in membrane
Implication: Models generalize well to unseen tissue regions
